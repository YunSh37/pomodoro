# 🍅 番茄钟 + 专注增强系统

> 不只是计时——把专注变成一场养成游戏。

一个用纯 Python 打造的 Windows 桌面番茄钟。除了经典的 25 分钟专注节奏，它把每一次坚持和每一次中断都变成了可视化的正负反馈：合成果实、晋升段位、解锁彩蛋。专注不再是苦差事。

## ✨ 为什么选它

大多数番茄钟只是一个倒计时。这个不一样——

| | 普通番茄钟 | 这个 |
|---|:---:|:---:|
| 计时 | ✅ | ✅ |
| 完成任务有奖励反馈 | ❌ | 🍎 合成果实 |
| 中断有代价感 | ❌ | 💔 心碎累积 |
| 长期坚持有成长 | ❌ | 🏆 段位晋升 |
| 数据回顾 | 基础统计 | 📊 图表 + 周报 |
| 满级彩蛋 | ❌ | 🎆 全屏特效 |

**每一次专注，你都在升级。**

## 🎮 核心玩法

### 果实合成

每完成一个番茄，获得一颗白色碎片。5 个合成升级：

```
白 → 绿 → 紫 → 红
```

红果实满 5 个即为巅峰，触发 **专注彩蛋** 🎆。

### 心碎系统

每次中断，获得一颗心碎。3 个合成升级：

```
💔 → 🤡 → 🖤
```

黑心满 5 个触发 **心碎彩蛋** 💔——算是一种黑色幽默。

### 段位晋升

连续专注天数决定你的段位：

```
青铜 → 白银 → 黄金 → 钻石 → 专注大师 → 番茄神
```

连续 41 天不断，你就是番茄神。

### 情绪追踪

每次完成番茄后记录此刻心情——从"已傻"到"巅峰"。系统根据你的情绪风格，给出不同语气的中肯建议。

## 📸 功能一览

- ⏱ **多档计时** — 5 / 10 / 15 / 25 / 45 / 60 分钟，自由切换
- 📋 **任务管理** — 添加、编辑、标记完成、软删除（可恢复）、永久删除
- 📊 **数据统计** — 今日 / 本周 / 本月专注时长柱状图 + 情绪饼图
- 📄 **周报导出** — 一键生成 PNG 周报，分享你的专注成果
- 🌓 **深色模式** — 自动跟随系统，也可手动切换
- 🎆 **满级彩蛋** — 全屏粒子特效，专注也能很燃

## 🚀 快速开始

```bash
# 克隆
git clone https://github.com/YunSh37/pomodoro.git
cd pomodoro

# 依赖（首次运行自动安装，也可手动）：
pip install matplotlib Pillow

# 启动
python main.py
```

> **环境要求**：Python 3.9+，Windows 10/11。

## 🧩 技术栈

- **Python 3.9+** — 标准库 tkinter 做 GUI，无重型框架
- **matplotlib** — 统计图表嵌入界面
- **Pillow** — 周报图片合成
- **JSON** — 单文件本地存储，数据完全在你自己手里
- **MVC 架构** — 模型 / 视图 / 控制器分离，易于二次开发
